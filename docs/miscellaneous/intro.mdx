# Miscellaneous

## What is Misc?

The Miscellaneous (aka **Misc**) category is the "everything else" bin of CTFs.
If a challenge doesn't fit into Web, Pwn, Rev or Crypto, it probably ends up here.
These challenges don't always require specific technical knowledge;
instead, they require some *imagination* to figure out what the challenge author wanted you to do.

In this section, we'll see some common Misc topics found in CTFs and explore tools that can make our lives easier.
Some Misc challenges involve *searching informations* (OSINT / Forensics) or "high-level" programming.
